# Public Guides Index

這份文件用來記錄站內公開 HTML 頁面的路徑與網址，方便後續新增品牌懶人包、教學頁、說明頁時統一管理。

## Base URL

`https://goodneighborpet.github.io/pet-commerce-link-hub/`

## Public Pages

| 類型 | 名稱 | 檔案路徑 | 公開網址 | 備註 |
| --- | --- | --- | --- | --- |
| 品牌懶人包 | GooToe 活力零食 | `guides/brands/gootoe.html` | `https://goodneighborpet.github.io/pet-commerce-link-hub/guides/brands/gootoe.html` | 狗零食挑選懶人包 |

## 建議新增規則

- 品牌懶人包：`guides/brands/<brand>.html`
- 教學頁：`guides/tutorials/<topic>.html`
- 說明頁：`guides/docs/<topic>.html`

## 新增頁面時記得同步

1. 建立對應的 `.html` 檔案
2. 補上這份 `guides/README.md`
3. 如果需要入口，再從 `index.html` 或其他頁面加上連結
